# azure-storage
☁️ Azure Storage Fundamentals

«Practical hands-on experience with Azure Storage Services using the Azure Portal.»

---

📖 Project Overview

This repository documents my Azure Storage learning journey and practical exercises completed in Microsoft Azure.

The project focuses on understanding and comparing the core Azure Storage services:

- 📦 Blob Storage
- 📁 Azure Files
- 📨 Queue Storage
- 🗄️ Table Storage

---

🎯 Learning Objectives

- Create and configure an Azure Storage Account
- Work with Blob Containers and file uploads
- Create and manage Azure File Shares
- Understand Queue Storage concepts
- Explore Table Storage
- Compare Azure Storage services and their use cases

---

🏗️ Azure Storage Architecture

Azure Storage Account
│
├── Blob Containers
│ └── Unstructured Files
│
├── File Shares
│ └── Shared Folders
│
├── Queues
│ └── Application Messages
│
└── Tables
    └── NoSQL Data

📨 Queue Storage

Purpose

Queue Storage enables asynchronous communication between applications.

Example

- Order Processing
- Background Tasks
- Message Handling

🗄️ Table Storage

Purpose

- Inventory Information

📊 Storage Comparison

Service| Data Type| Typical Use Case
Blob Storage| Unstructured Data| Documents, Images, Backups
Azure Files| Shared Files| Team Collaboration
Queue Storage| Messages| Application Communication
Table Storage| NoSQL Data| Metadata & Records
